Any `Contextable` API argument can be passed as a function whose single parameter is a `Context` object. `Context` exposes all the methods available for the current `table`, `assertion`, or `operation`.

This is useful when you need to use context methods within an argument.

## Example: Use `ref()` in JavaScript

When defining a dataset via the JavaScript API, the `query()` method takes a `Contextable<string>` argument, and can be called in two ways:

```js
publish("example_simple").query("SELECT 1 AS TEST");

publish("example_context").query(context => `SELECT * FROM ${context.ref("example_simple")}`);
```
